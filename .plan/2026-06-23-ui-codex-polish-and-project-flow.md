## 目标

对齐用户最新提出的 Codex 风格桌面 UI 调整，重点补齐真实可用的项目流、顶部菜单栏、右上环境浮窗、底部控制区和设置项行为。

## 本轮范围

1. 顶栏与窗口控制同排，补齐环境按钮图标与对齐。
2. 项目侧栏去掉无意义按钮，增加项目 hover 新对话动作，并让新对话绑定项目目录。
3. 输入区调整宽高比例，补 `Shift+Tab` 切换计划模式，计划模式状态在权限栏右侧可见。
4. 权限、模型、发送按钮、下拉浮窗统一图标尺寸、文字基线和层级。
5. 修复字体缩放对主要界面不生效的问题，统一滚动条视觉。
6. 启动本地桌面应用验证关键 UI 交互。

## 验证点

- 顶部菜单与最小化/最大化/关闭按钮在同一行且基线一致。
- 右上环境按钮可打开浮窗，浮窗不被输入区遮挡。
- `Shift+Tab` 可切换计划模式，底部出现计划状态标识。
- 项目 hover 可新建该目录下对话，新会话自动绑定该项目。
- 设置中的字体大小调整能实时影响顶部、侧栏、对话区和浮窗。
